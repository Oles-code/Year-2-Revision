## 1. σ-Algebras, Algebras, and Borel Sets

**Exam relevance:** True/False questions test whether you can identify σ-algebras, determine Borel measurability, and use generating sets. Dedicated questions on algebras vs σ-algebras appeared in 2022 Q2.

### 1.1 Key Definitions

A family **A** ⊂ 2^X is an **algebra** if it contains X, is closed under complements, and closed under *finite* unions. It is a **σ-algebra** if the closure extends to *countable* unions.

The **Borel σ-algebra** B(ℝⁿ) is σ(τ), where τ is the collection of open sets in ℝⁿ. Equivalently, B(ℝ) is generated by any of these families: open intervals (a,b), closed intervals [a,b], half-open intervals, or rays (-∞, a).

**Key fact (tested in 2023 Q1a):** Closed sets like [0,1] belong to B(ℝ). You prove this by writing [0,1] = ∩ₙ (-1/n, 1+1/n), a countable intersection of open sets.

### 1.2 Algebra That Is Not a σ-Algebra

**Classic example (tested 2022 Q2):** Let N = {1,2,3,...} and A = {A ⊂ N : A finite or Aᶜ finite}. Then:

- **A is an algebra:** N ∈ A (Nᶜ = ∅ is finite). Complements clearly stay in A. For finite unions: if all Aᵢ are finite, so is their union; if some Aᵢ has finite complement, then (∪Aᵢ)ᶜ ⊂ Aᵢᶜ is finite.
- **A is not a σ-algebra:** Take Aₙ = {2n}. Each is finite, so Aₙ ∈ A. But ∪Aₙ = {even numbers}, which is infinite with infinite complement, so ∪Aₙ ∉ A.

### 1.3 Generated σ-Algebra: σ(C)

For any collection C ⊂ 2^X, σ(C) is defined as the intersection of all σ-algebras containing C. This is the *smallest* σ-algebra containing C.

**Useful fact (Sheet 1, Q4):** For any A ∈ σ(C), there exists a *countable* sub-collection D ⊂ C such that A ∈ σ(D). This is proven by showing the collection of all such "countably generated" sets is itself a σ-algebra containing C.

## 2. Measures and Pre-Measures

**Exam relevance:** Every paper includes questions on measure properties. The 2022 Q2c tests whether a given function is a pre-measure. The 2025 Q1a tests basic measure axioms.

### 2.1 Definition of a Measure

A measure μ on (X, A) satisfies: (i) μ(∅) = 0, and (ii) **σ-additivity:** for pairwise disjoint (Aₖ) ⊂ A, μ(∪Aₖ) = Σ μ(Aₖ).

### 2.2 Fundamental Properties

**Proposition 1.5 (know these cold — tested implicitly everywhere):**

1. **Monotonicity:** A ⊂ B ⟹ μ(A) ≤ μ(B).
2. **Continuity from below:** Aₖ ↑ A ⟹ μ(A) = lim μ(Aₖ).
3. **Continuity from above:** Aₖ ↓ A and **μ(A₁) < ∞** ⟹ μ(A) = lim μ(Aₖ).
4. **σ-subadditivity:** μ(A) ≤ Σ μ(Aₖ) whenever A ⊂ ∪Aₖ.

⚠️ **The finiteness condition in (3) is essential.** Counter-example: counting measure on ℕ, Aₖ = {k, k+1, ...}. Then Aₖ ↓ ∅ but μ(Aₖ) = ∞ for all k.

### 2.3 Pre-Measures and When They Fail

A **pre-measure** is a function μ̃ on an algebra A satisfying μ̃(∅) = 0 and σ-additivity *whenever the countable union stays in A*.

**Testing whether something is a pre-measure (2022 Q2c):** The function λ defined on the algebra A from §1.2 by λ(A) = Σₙ∈A 1/n² (A finite) or λ(A) = 2 - Σₙ∈Aᶜ 1/n² (Aᶜ finite) is **not** a pre-measure. The proof: N = ∪ₖ {k} ∈ A and λ(N) = 2, but Σₖ λ({k}) = Σ 1/k² = π²/6 ≠ 2.

### 2.4 Absolute Continuity of Measures

**Tested in every single exam paper.** ν ≪ μ means: μ(A) = 0 ⟹ ν(A) = 0.

If ν has a density f with respect to μ (i.e. ν(A) = ∫_A f dμ), then ν ≪ μ. The converse is the **Radon-Nikodym Theorem** (§8).

**Common exam trap (2023 Q1c, 2025 Q1d):** dμ = 1_{[0,1]} dλ and dν = ½·1_{[0,2]} dλ. Is ν ≪ μ? **No**, because μ([1,2]) = 0 but ν([1,2]) = ½ > 0. Is μ ≪ ν? **Yes**, because if ν(A) = 0 then λ(A ∩ [0,2]) = 0, so λ(A ∩ [0,1]) = 0, so μ(A) = 0.

## 3. Construction of the Lebesgue Measure (Hahn-Carathéodory)

**Exam relevance:** The 2025 Q2 is an entire 20-mark question on the outer measure construction from a specific function f. Understanding this pipeline is essential.

### 3.1 The Construction Pipeline

```
Pre-measure μ̃ on algebra A
        ↓ Step 1: formula (1.16)
Outer measure μ* on 2^X
        ↓ Step 2: restrict to Σ (Carathéodory σ-algebra)
Measure μ on (X, Σ)
```

**Step 1** defines μ*(A) = inf { Σ μ̃(Kⱼ) : Kⱼ ∈ A, A ⊂ ∪ Kⱼ }.

**Step 2** defines Σ = { A ⊂ X : μ*(B) = μ*(B ∩ A) + μ*(B \ A) for all B ⊂ X }.

### 3.2 Hahn-Carathéodory Extension Theorem

**Theorem 1.13:** Given pre-measure μ̃ on algebra A, the construction above yields a measure space (X, Σ, μ) with A ⊂ Σ and μ|_A = μ̃. If μ̃ is **σ-finite**, the extension is unique (Theorem 1.14).

### 3.3 Proving μ*(⟨a,b)) = μ(⟨a,b)) for a Specific f (2025 Q2 Style)

The 2025 exam asked this for a non-decreasing, left-continuous function f. The key steps:

- **μ* ≤ μ:** Choose I₀ = [a,b), Iⱼ = ∅ for j ≥ 1 in the infimum defining μ*.
- **μ ≤ μ*:** This is the hard direction. Given a cover [a,b) ⊂ ∪ [aⱼ,bⱼ), use left-continuity of f and compactness: shrink [a,b) slightly to [a,b'] (with f(b) - f(b') < ε/2), enlarge each [aⱼ,bⱼ) to (aⱼ',bⱼ), then extract a finite subcover by Heine-Borel and use monotonicity of f. This is a classic ε-argument.

## 4. Measurable Functions and Integration

### 4.1 Measurability

f: X → ℝ is measurable if f⁻¹(A') ∈ A for all A' ∈ B(ℝ). Equivalently, it suffices to check f⁻¹((-∞, a)) ∈ A for all a ∈ ℝ.

**Key stability results (Theorem 2.4):** Sums, products, absolute values, max, min of measurable functions are measurable. Pointwise limits (sup, inf, lim sup, lim inf) of measurable sequences are measurable.

### 4.2 Integration in Three Steps

| Step | Domain | Definition |
|------|--------|------------|
| 1 | Simple functions s = Σ αᵢ 1_{Aᵢ} | ∫ s dμ = Σ αᵢ μ(Aᵢ) |
| 2 | Non-negative measurable f ≥ 0 | ∫ f dμ = sup { ∫ g dμ : g simple, 0 ≤ g ≤ f } |
| 3 | Integrable f (i.e. ∫|f| dμ < ∞) | ∫ f dμ = ∫ f⁺ dμ - ∫ f⁻ dμ |

where f⁺ = max(f, 0) and f⁻ = max(-f, 0).

### 4.3 Properties of the Integral

- **Monotonicity:** f ≤ g μ-a.e. ⟹ ∫ f dμ ≤ ∫ g dμ.
- **Triangle inequality:** |∫ f dμ| ≤ ∫ |f| dμ.
- **Linearity:** ∫(αf + βg) dμ = α∫f dμ + β∫g dμ.
- **Vanishing:** f = 0 μ-a.e. ⟺ ∫|f| dμ = 0 (for f ≥ 0 measurable).

## 5. The Big Three Convergence Theorems ⭐

**These are the most important results in the course for exam purposes.** Every single paper (2022–2025) has at least one question requiring you to apply MCT, Fatou, or DCT — often as the core of a multi-part problem worth 12–16 marks.

### 5.1 Monotone Convergence Theorem (MCT)

**Theorem 2.17(ii):** If fₙ: X → [0,∞] are measurable with fₙ ↑ f (i.e. fₙ(x) ≤ fₙ₊₁(x) and fₙ(x) → f(x) for all x), then:

∫ f dμ = lim_{n→∞} ∫ fₙ dμ

⚠️ Requires **non-negativity** and **monotone increase**. Without non-negativity, it can fail: fₙ = -1/n, f = 0 gives ∫ fₙ dλ = -∞ but ∫ f dλ = 0.

### 5.2 Fatou's Lemma

**Theorem 2.22:** If fₙ: X → [0,∞] are measurable, then:

∫ (lim inf fₙ) dμ  ≤  lim inf ∫ fₙ dμ

**The inequality can be strict:** Take fₙ alternating between 1_E and 1_{Eᶜ}. Then lim inf fₙ = 0 everywhere, so the left side is 0, but lim inf ∫ fₙ = min{μ(E), μ(Eᶜ)} > 0.

**Key application (2023 Q4a, 2024 Q3a):** Showing f ∈ Lp when fₙ → f a.e. and sup ‖fₙ‖_Lp < ∞:

‖f‖ᵖ = ∫ lim |fₙ|ᵖ dμ ≤ lim inf ∫ |fₙ|ᵖ dμ ≤ sup ‖fₙ‖ᵖ_Lp < ∞

### 5.3 Dominated Convergence Theorem (DCT) ⭐⭐

**Theorem 2.24:** If fₙ → f μ-a.e., |fₙ| ≤ g for some g ∈ L¹(μ), then:

∫ |fₙ - f| dμ → 0  (and in particular ∫ fₙ dμ → ∫ f dμ)

**Proof sketch:** Apply Fatou to 2g - |fₙ - f| ≥ 0, giving ∫ 2g ≤ lim inf ∫(2g - |fₙ - f|) = ∫ 2g - lim sup ∫|fₙ - f|, hence lim sup ∫|fₙ - f| ≤ 0.

### 5.4 How to Apply DCT on an Exam — A Checklist

This is the single most common exam technique. When asked to evaluate lim ∫ fₙ dλ:

1. **Find the pointwise limit:** Determine f(x) = lim fₙ(x).
2. **Find a dominating function g ∈ L¹:** Show |fₙ(x)| ≤ g(x) for all n and x.
3. **Verify g is integrable.**
4. **Conclude:** lim ∫ fₙ = ∫ f by DCT.

### 5.5 Worked Example: 2022 Q3b

Evaluate lim_{n→∞} ∫₀^∞ [n sin(x/n)] / [x(1+x²)] dλ.

**Step 1 — pointwise limit:** For fixed x > 0, n sin(x/n) / x = sin(x/n)/(x/n) → 1, so fₙ(x) → 1/(1+x²).

**Step 2 — dominating function:** Need |fₙ(x)| ≤ g(x) ∈ L¹.

- If x ≤ n: |sin(y)/y| ≤ C for y ∈ [0,1] (it's a bounded continuous function on [0,1]), so |fₙ(x)| ≤ C/(1+x²).
- If x > n: |n sin(x/n)| ≤ n, and n/[x(1+x²)] ≤ 1/(1+x²).

So g(x) = C/(1+x²) works, and g ∈ L¹(0,∞) since ∫₀^∞ dx/(1+x²) = π/2.

**Step 3 — conclude:** lim ∫ fₙ = ∫₀^∞ 1/(1+x²) dx = π/2.

### 5.6 Common Dominating Functions

| Integrand type | Typical dominator |
|----------------|-------------------|
| sin(stuff)/x · decay | Use \|sin\| ≤ min(1, \|arg\|), then bound by integrable decay |
| Bounded × indicator | M · 1_{bounded set} |
| Rational in x with polynomial decay | C/(1 + x²) or C/x^α for suitable α |

## 6. Modes of Convergence

**Exam relevance:** True/False Q1 parts test these in every paper. Understanding the relationships and counterexamples is essential.

### 6.1 Definitions

| Mode | Notation | Definition |
|------|----------|------------|
| Pointwise a.e. | fₙ → f μ-a.e. | μ({x : fₙ(x) ↛ f(x)}) = 0 |
| In measure | fₙ →^μ f | ∀ε > 0: μ({|fₙ - f| > ε}) → 0 |
| In Lᵖ | fₙ →^{Lp} f | ∫ |fₙ - f|ᵖ dμ → 0 |
| Uniformly | fₙ ⇒ f | sup_x |fₙ(x) - f(x)| → 0 |

### 6.2 Implications (When μ(X) < ∞)

```
Uniform  ⟹  Pointwise a.e.  ⟹  In measure
                                    ⇑
                              L¹ convergence
```

And: convergence in measure ⟹ a subsequence converges a.e. (Theorem 2.28).

**None of the reverse implications hold in general.** Key counterexamples:

- **In measure ⟹̸ a.e.:** The "typewriter sequence" fₙ = 1_{Iₙ} where Iₙ cycles through [0,1] with shrinking width. Converges in measure to 0 but diverges everywhere (Sheet 4, Q6b).
- **a.e. ⟹̸ in measure (when μ(X) = ∞):** fₙ = 1_{[n,n+1]} → 0 a.e. but λ({|fₙ| > ½}) = 1 (Sheet 5, Q8c).

### 6.3 Checking Convergence in Measure (Exam Standard)

To show fₙ →^μ 0, show that for all ε > 0, λ({|fₙ| > ε}) → 0. A common trick uses |sin(x)| ≤ |x|:

**Example (2023 Q2a, 2024 Q1c):** fₙ(x) = n^{1/4} sin(x/n^{5/4}) · 1_{|x|≤n^{3/4}}.

λ({|fₙ| > ε}) ≤ λ({|x| ≤ n^{3/4} and |x|/n > ε}) → 0 since the set becomes empty for large n.

### 6.4 Checking Lp Convergence

‖fₙ‖²_L² = ∫ |fₙ|² dλ. Compute this explicitly and check whether it → 0.

**Example (2022 Q1a, 2025 Q1c):** fₙ(x) = n^{1/3} · 1_{(2/n, 3/n)}(x). Then ‖fₙ‖²₂ = n^{2/3} · (1/n) = n^{-1/3} → 0. ✓

## 7. Fubini's Theorem and Product Measures

**Exam relevance:** A Fubini question appears in 2022 Q4a, 2023 Q3, and 2025 Q3 — usually worth 12–20 marks.

### 7.1 Statement

**Theorem 3.5 (Fubini):** For σ-finite (Xᵢ, Aᵢ, μᵢ), if f ≥ 0 or f ∈ L¹(μ₁ ⊗ μ₂), then:

∫ f d(μ₁ ⊗ μ₂) = ∫ ( ∫ f(x₁,x₂) dμ₂ ) dμ₁ = ∫ ( ∫ f(x₁,x₂) dμ₁ ) dμ₂

### 7.2 When Does Fubini Apply?

- **f ≥ 0:** Always applies (the iterated integrals might be ∞, but they exist and agree).
- **f changes sign:** Must first verify f ∈ L¹. Typically you show |f| ∈ L¹ by applying Fubini to |f| (which is ≥ 0, so always valid).

### 7.3 The Contrapositive Trick (2023 Q3b)

If you compute the two iterated integrals and they **disagree**, then f ∉ L¹. This is because if f were in L¹, Fubini would force the two to be equal.

**Example (2023 Q3b):** f(x,y) = (x-y)/(x+y)³ on E = [1,∞)². By symmetry f(x,y) = -f(y,x), so the two iterated integrals are negatives of each other. Computing one gives ½, the other gives -½. Hence f ∉ L¹.

### 7.4 Exam Strategy for Fubini Problems

1. **If f changes sign:** First bound |f| by something simpler, show that simpler thing is L¹ via Fubini (it's ≥ 0), then apply Fubini to f itself.
2. **If asked to "use Fubini to show ‖G‖_{L¹} ≤ C":** Apply Fubini to |G| (non-negative case), integrate in the easier variable first.
3. **Computing ∫₀^∞ e^{-xy} dy = 1/x** (for x > 0) appears frequently. Justify by MCT: ∫₀^n e^{-xy} dy ↑ 1/x.

## 8. Lp Spaces

**Exam relevance:** Testing Lp membership appears in every paper (Q1). Deeper Lp questions appear in 2022 Q4b (non-separability of L∞), 2023 Q4 (Brezis-Lieb style), and 2024 Q3 (Markov/tail bounds).

### 8.1 Key Inequalities

**Markov's inequality:** For f ≥ 0 measurable and M > 0:

μ({f ≥ M}) ≤ (1/M) ∫ f dμ

**Hölder's inequality:** For 1/p + 1/q = 1:

‖fg‖₁ ≤ ‖f‖_p · ‖g‖_q

**Minkowski's inequality:** ‖f + g‖_p ≤ ‖f‖_p + ‖g‖_p

### 8.2 Checking Lp Membership (Exam Standard)

For f(x) = |x|^{-α} on X = (0,1): f ∈ Lp(0,1) ⟺ αp < 1.

For f(x) = |x|^{-α} on X = (1,∞): f ∈ Lp(1,∞) ⟺ αp > 1.

**Consequence:** |x|^{-α} is never in any Lp(0,∞).

**Exam technique for "is f ∈ Lp?" questions:** Split the domain at problem points (0 and ∞), check integrability of |f|^p near each separately.

### 8.3 Tail Bounds and Layer Cake (2024 Q3)

**Layer cake formula:** ‖f‖ᵖ_p = ∫₀^∞ p t^{p-1} μ(|f| ≥ t) dt.

**Proof:** Apply Fubini to the non-negative function p t^{p-1} 1_{|f(x)| ≥ t}, integrating first in t:

∫_X ∫₀^{|f(x)|} p t^{p-1} dt dμ(x) = ∫_X |f(x)|ᵖ dμ(x)

**Markov for Lp (2024 Q3a):** t^p μ(|f| ≥ t) ≤ ‖f‖ᵖ_p.

### 8.4 L∞ Is Not Separable (2022 Q4b)

The family {fₜ = 1_{[0,t]}: 0 < t < 1} satisfies ‖fₜ - fₛ‖_{L∞} = 1 for all s ≠ t. Any ball of radius < ½ around fₜ contains at most one element of this uncountable family. Hence no countable dense subset exists.

## 9. Vitali's Convergence Theorem

**Exam relevance:** Explicitly tested in 2024 Q2b. The concept of uniformly absolutely continuous integrals also appears implicitly.

### 9.1 Uniformly Absolutely Continuous Integrals

A family F ⊂ L¹(μ) has uniformly absolutely continuous integrals if:

∀ε > 0, ∃δ > 0, ∀f ∈ F, ∀A ∈ A : μ(A) < δ ⟹ ∫_A |f| dμ < ε

### 9.2 Vitali's Theorem (μ(X) < ∞)

fₙ →^{L¹} f  ⟺  fₙ →^μ f  AND  {fₙ} has uniformly abs. cont. integrals.

This is **stronger than DCT** — it gives a necessary *and* sufficient condition.

## 10. Egorov's Theorem and Lusin's Theorem

**Exam relevance:** Lusin's theorem was tested in 2023 Q1d(ii). Egorov is used as a tool in proving Vitali's theorem and in the 2025 Q4 question.

### 10.1 Egorov's Theorem

If λ(Ω) < ∞ and fₖ → f a.e. on Ω, then ∀δ > 0, ∃ compact F ⊂ Ω with λ(Ω \ F) < δ and fₖ → f **uniformly** on F.

*Idea:* "Almost-everywhere convergence is nearly uniform convergence."

### 10.2 Lusin's Theorem

If λ(Ω) < ∞ and f: Ω → ℝ is measurable, then ∀δ > 0, ∃ compact F ⊂ Ω with λ(Ω \ F) < δ and f|_F is **continuous**.

*Idea:* "Every measurable function is nearly continuous."

## 11. The Cantor Set and Devil's Staircase

**Exam relevance:** The entire 2024 Q4 (20 marks) is about the Cantor set and its associated function.

### 11.1 Construction

C₀ = [0,1]. Cₖ is obtained by removing the open middle third of each interval in Cₖ₋₁. The Cantor set C = ∩ₖ Cₖ.

### 11.2 Key Properties

- **C is uncountable:** Each point maps bijectively to a sequence in {0,1}^ℕ (left/right choice at each step).
- **λ(C) = 0:** λ(Cₖ) = (2/3)^k → 0.
- **C is closed** (intersection of closed sets), **nowhere dense**, and **perfect** (every point is a limit point).

### 11.3 The Cantor Function (Devil's Staircase)

Define μₖ with density λ(Cₖ)⁻¹ · 1_{Cₖ} and fₖ(x) = μₖ((0,x]). Then:

- **|fₖ₊₁(x) - fₖ(x)| ≤ 2⁻ᵏ** — so the fₖ converge uniformly to a continuous f.
- **f(0) = 0, f(1) = 1.**
- **f'(x) = 0 a.e.** — because f is locally constant on [0,1] \ C, and λ(C) = 0.

This gives an example of a continuous, non-decreasing function from [0,1] onto [0,1] whose derivative is zero almost everywhere.

## 12. Lebesgue Differentiation Theorem

**Exam relevance:** Used as background in 2025 Q4b for Riemann sum convergence.

### 12.1 Statement

For f ∈ L¹_loc(ℝⁿ), for a.e. x ∈ ℝⁿ:

f(x) = lim_{r→0} (1/λ(B(x,r))) ∫_{B(x,r)} f(y) dy

The proof uses the **Hardy-Littlewood maximal inequality:** λ({f* > a}) ≤ (5ⁿ/a) ‖f‖_{L¹}, and the density of C⁰_c in L¹.

## 13. Radon-Nikodym and Lebesgue Decomposition

### 13.1 Lebesgue Decomposition

Any σ-finite ν can be uniquely written as ν = ν_s + ν_{ac} where ν_s ⊥ μ (singular) and ν_{ac} ≪ μ (absolutely continuous, with density).

### 13.2 Radon-Nikodym Theorem

ν ≪ μ ⟺ ν = fμ for some measurable f ≥ 0. The function f = dν/dμ is the **Radon-Nikodym derivative**.

## Appendix: Quick Reference for True/False Questions

| Statement Type | Key Test |
|---|---|
| "fₙ converges uniformly" | Check sup_x \|fₙ(x)\| → 0 |
| "fₙ converges in measure" | Check λ({\|fₙ\| > ε}) → 0, often using \|sin x\| ≤ \|x\| |
| "fₙ converges in Lp" | Compute ∫\|fₙ\|ᵖ explicitly |
| "f ∈ Lp(X)" | Check ∫\|f\|ᵖ < ∞ by splitting near singularities and at ∞ |
| "ν ≪ μ" | Find a set A with μ(A)=0 and check if ν(A)=0 |
| "A is in the Borel σ-algebra" | Write A as countable union/intersection of open/closed sets |
| "μ is a (pre-)measure" | Check σ-additivity (when applicable); try to find a violation |
| "A measure m with given values exists" | Check inclusion-exclusion: m(A∪B) = m(A)+m(B)-m(A∩B) |
